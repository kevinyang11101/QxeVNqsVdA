# 旅游景点系统介绍 python1251

## 项目概述


## 技术栈

- 后端框架：Django（使用Python开发）
- 前端框架：Vue.js
- 数据库：MySQL 5.7或更高版本
- 开发工具：
  - Python版本：3.7/3.8
  - 数据库工具：Navicat 11（推荐）
  - 集成开发环境：PyCharm/VS Code

## 功能模块

- 用户偏好设置
- 旅游线路推荐
- 系统管理

## 系统角色

- 普通用户
- 管理员

## 运行环境

- Python 3.7/3.8
- MySQL 5.7或更高版本
- Node.js（前端Vue.js框架运行依赖）

## 部署步骤

1. 安装Python运行环境
2. 安装MySQL数据库，并创建对应数据库
3. 部署后端服务（Django）
4. 部署前端服务（Vue.js）
5. 启动服务，进行测试

## 目录结构

```
project/
│
├── backend/                # 后端代码目录
│   ├── apps/               # Django应用目录

│   └── ...
│
├── frontend/               # 前端代码目录
│   ├── src/                # Vue.js源代码目录
│   ├── public/             # Vue.js公共文件目录
│   └── ...
│
└── ...
```

## 核心亮点

- 基于协同过滤算法，为用户提供精准的旅游线路推荐。
- 使用Django框架，提供稳定可靠的后端服务。
- 前端采用Vue.js框架，实现响应式和组件化的界面设计。
